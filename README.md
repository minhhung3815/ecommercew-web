# Ecommerce-Web
E-commerce web application that sells technology products such as mobile phone, tablet, laptop, screen,...

## Summary
The project contains 3 major modules:
* **Front-end Module**: Using **ReactJS** library to build the front-end of the web application.
* **Back-end Module**: Using **ExpressJS** framework to build the API to communicate with front-end and implement logical businesses in the web application.
* **Database Module**: Using **MongoDB** for simple storage.

## Problems:
The problem can be described as:
* User goes to the web application to buy his or her desired products.
* After sign in to the web, user can make order for any products that are available on the web.
* User needs to make payment via Paypal gateway to complete the ordering process.

## Requirements
* Linux, OS X or Windows.
* NodeJS v16.0.3.

## Installation:
#### Install Back-end Modules:
```bash
#Install backend modules
npm i --save
```
#### Install Front-end Modules:
```bash
#Come to folder contain frontend source
cd frontend
#Install frontend modules
npm i --save
```

## Run Application:
#### Run Back-end:
````bash
npm run dev
````
#### Run Front-end:
```bash
cd frontend
npm start
```
